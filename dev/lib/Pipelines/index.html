Pipelines

Pipelines is a library designed to generate and evaluate data analysis pipelines.

Transformation interface

Pipelines.AbstractCard — Type
abstract type AbstractCard end

Abstract supertype to encompass all possible filters.

Current implementations:

  • RescaleCard,
  • SplitCard,
  • GLMCard.

Replace table target in the database repo.db with the outcome of executing the card on the table source.

Cards

Pipelines.RescaleCard — Type
struct RescaleCard <: AbstractCard
    method::String
    by::Vector{String} = String[]
    columns::Vector{String}
    suffix::String = "rescaled"
end

Card to rescale of one or more columns according to a given method. The supported methods are

  • zscore,
  • maxabs,
  • minmax,
  • log,
  • logistic.

The resulting rescaled variable is added to the table under the name "$(originalname)_$(suffix)".

source
Pipelines.SplitCard — Type
struct SplitCard <: AbstractCard
    method::String
    order_by::Vector{String}
    by::Vector{String} = String[]
    output::String
    p::Float64 = NaN
    tiles::Vector{Int} = Int[]
end

Card to split the data into two groups according to a given method.

Currently supported methods are

  • tiles (requires tiles argument, e.g., tiles = [1, 1, 2, 1, 1, 2]),
  • percentile (requires p argument, e.g. p = 0.9).
